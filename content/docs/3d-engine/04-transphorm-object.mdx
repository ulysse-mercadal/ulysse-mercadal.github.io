---
title: Transform an object
description: Transform an object on the viewport
---

After creating an object with `addObject()`, you can dynamically update its position, rotation, or scale using the following public methods.
These method are parts of the **`TransformController`**, accessible via `Renderer3D::getTransformController()`.

---

## Available Methods

### `setObjectPosition`

Sets the world-space position of an instance.

```cpp
bool TransformController::setObjectPosition(
    const std::string& modelPath,
    int instanceId,
    const glm::vec3& position
);
```

**Example**:
```cpp
int id = renderer.getAssetController().addObject("assets/indien-qui-danse.fbx", {0,0,0});
renderer.getTransformController().setObjectPosition("assets/indien-qui-danse.fbx", id, {5.0f, 1.0f, -3.0f});
```

---

### `setObjectRotation`

Sets absolute Euler rotation in **radians** (YXZ convention).

```cpp
bool TransformController::setObjectRotation(
    const std::string& modelPath,
    int instanceId,
    const glm::vec3& rotation
);
```

**Example**:
```cpp
int id = renderer.getAssetController().addObject("assets/indien-qui-danse.fbx", {0,0,0});
renderer.getTransformController().setObjectRotation("assets/indien-qui-danse.fbx", id, {0.0f, glm::radians(90.0f), 0.0f});
```

---

### `setObjectScale`

Sets local scale per axis (or uniform if using `glm::vec3(s)`).

```cpp
bool TransformController::setObjectScale(
    const std::string& modelPath,
    int instanceId,
    const glm::vec3& scale
);
```

**Example**:
```cpp
int id = renderer.getAssetController().addObject("assets/indien-qui-danse.fbx", {0,0,0});
renderer.getTransformController().setObjectScale("assets/indien-qui-danse.fbx", id, {2.0f, 2.0f, 2.0f}); // uniform
renderer.getTransformController().setObjectScale("assets/indien-qui-danse.fbx", id, {1.0f, 2.0f, 0.5f}); // non-uniform
```

---

### `setObjectTransform`

Updates position, rotation, and scale in one call (more efficient).

```cpp
bool TransformController::setObjectTransform(
    const std::string& modelPath,
    int instanceId,
    const glm::vec3& position,
    const glm::vec3& rotation,
    const glm::vec3& scale
);
```

## Return Value

All methods return:
- `true` if the model and instance exist and the transform was updated,
- `false` if `modelPath` is not loaded or `instanceId` is out of bounds.

## Behavior Notes

- The new transform is applied **starting from the next frame** (used in `cullAndSortObjects`).

## Usage in Game Loop

```cpp
// In initialization
int playerModelId = renderer.getAssetController().addObject("assets/indien-qui-danse.fbx", {0,0,0});

// In update loop
glm::vec3 newPos = playerPosition + movementInput;
renderer.getTransformController().setObjectPosition("assets/indien-qui-danse.fbx", playerModelId, newPos);
```
